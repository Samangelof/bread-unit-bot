# Бот для расчета хлебных единиц (ХЕ)

Этот бот предназначен для расчета хлебных единиц в продуктах. Хлебные единицы используются для оценки содержания углеводов в продуктах и их влияния на уровень глюкозы в крови у людей с сахарным диабетом.

## Установка
1. Установить необходимые зависимости:
    ```bash
      pip install -r requirements.txt
    ```
2. Задать токен бота в переменную TOKEN в файле main.py


## Использование
1. Запустить бота:
    ```bash
      pip install -r requirements.txt
    ```
2. Для начала работы написать боту /start и следовать инструкциям


## Команды
- /start - начало работы с ботом
- /calculate_he [название продукта] [количество в граммах] - расчет ХЕ в указанном продукте
- /about_bot - информация о боте
